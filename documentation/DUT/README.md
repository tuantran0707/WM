# Devices Under Test (DUT)

This directory contains EOL testing documentation for all NubeIO devices.

## Structure

### GEN 1 (Generation 1)
- **MicroEdge** - First generation edge device

### GEN 2 (Generation 2)
- **ACB-M** - Air Conditioning Controller Board
- **ZC-LCD** - Zone Controller with LCD display
- **ZC-Controller-Damper** - Zone Controller for damper control
- **Droplet** - IoT sensor device

### Support Boards
- **AC-Connect** - Device testing support board
- **ZC-Input-14In-8out** - Support board for reading ZoneController damper outputs

## Documentation Guidelines

Each device folder should contain:
- Step-by-step testing procedures
- Connection diagrams
- Expected test results
- Troubleshooting guide
- Firmware version requirements
